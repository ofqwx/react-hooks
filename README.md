# react-hooks
Simple daily basis react hooks

## Installation

With yarn:
```
yarn add @ofqwx/react-hooks
```
or npm:
```
npm i --save @ofqwx/react-hooks
```

If you prefer, you can install hooks individually:

```
yarn add @ofqwx/hookName
```
or npm:
```
npm i --save @ofqwx/hookName
```
